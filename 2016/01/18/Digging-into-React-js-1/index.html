Digging into React.js(1)

First day!

So, before everything begin, install & config webpackfor React & ES6 is literally a pain in the ass. Fortunately, some one create a lib called Turris.js would help. install it by npm install -g yo generator-turris (possibly require latest version of node & npm) then type yo turris to create a default project folder. Also, I am using Material-UI, so also have to install too much dependency. Anyway, after a while, I finished configure these basic stuff.

Ok After working a while. Basically, I got a clear idea about how it works. I have no idea how could I know that. And please please please don’t expect me to tell about the basic about React.js too much. All I could say is that the official docs suck. I learn it faster by just reading the code other people worked. Actually, it’s quite straight forward.
